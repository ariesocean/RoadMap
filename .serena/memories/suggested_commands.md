# Suggested Commands for RoadMap Project

## Development Commands
```bash
cd roadmap-manager
npm run dev              # Start Vite dev server (port 1430)
npm run build            # TypeScript compile + Vite build
npm run preview          # Preview production build
npm run opencode:server  # Start OpenCode server on port 51432
```

## System Commands (Darwin/macOS)
- `ls`, `cd`, `pwd` - standard Unix commands
- `git` - version control
- `npm` - package management

## Testing
- Uses Vitest for testing (see .vite/vitest/ in node_modules)
